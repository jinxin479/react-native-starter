# 技术栈
- react-native
- redux 状态管理
- redux-saga  异步任务管理
- React Navigation 导航
- CodePush 代码热更新
- redux-persist 持久化状态
# 项目结构介绍

```
├── src                      # js代码源文件
│   ├── constants            # 常量
│   ├── components           # UI组件
│   ├── srceens              # 以屏为单位的容器组件
│   ├── navigation           # 主页面导航
│   ├── assets               # 资源文件
│   ├── api                  # 接口
│   ├── utils                # 工具函数

```
